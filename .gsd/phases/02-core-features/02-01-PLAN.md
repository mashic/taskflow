<task type="auto">
  <name>02-01-T3: Create BoardsService and BoardsController</name>
  <files>
    apps/api/src/boards/boards.service.ts
    apps/api/src/boards/boards.controller.ts
    apps/api/src/boards/boards.module.ts
    apps/api/src/app.module.ts
  </files>
  <action>
    1. Create BoardsService:
       - Inject BoardsRepository
       - create(dto, userId): creates board with ownerId = userId
       - findAllForUser(userId): returns user's boards
       - findOne(id, userId): returns board if owned by user, else throw NotFoundException
       - update(id, dto, userId): updates if owned by user
       - remove(id, userId): soft deletes if owned by user
    
    2. Create BoardsController with @Controller('boards'):
       - @UseGuards(JwtAuthGuard) on controller level
       - @Post() create(@Body() dto, @Request() req): returns created board
       - @Get() findAll(@Request() req): returns user's boards
       - @Get(':id') findOne(@Param('id') id, @Request() req): returns single board
       - @Patch(':id') update(@Param('id') id, @Body() dto, @Request() req): updates board
       - @Delete(':id') remove(@Param('id') id, @Request() req): soft deletes board
    
    3. Create BoardsModule:
       - imports: [PrismaModule]
       - providers: [BoardsService, BoardsRepository]
       - controllers: [BoardsController]
       - exports: [BoardsService]
    
    4. Import BoardsModule in AppModule
  </action>
  <verify>
    `pnpm --filter api build` compiles successfully
    Module imports resolve correctly
  </verify>
  <done>BoardsModule integrated into app with REST endpoints</done>
